- This is a vscode plugin for writing notes for personal use. 
- It has a similar structure to Obsidian/Logseq with a daily note and links
    - Every daily note is in a file named YYYY-MM-DD.md
- Date format is YYYY-MM-DD, but could also be YYYY-MM or just YYYY
- Notes are written in Markdown with Markdown wiki-links ([[example]])
    - every link can have a file/page, which will be stored in example.md
    - every individual link with its context is a linklocation
